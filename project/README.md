# AntzBoardgame Website

A modern, responsive single-page website for AntzBoardgame cafe in Miri, Sarawak.

## Features

- **Responsive Design**: Mobile-first approach with beautiful layouts
- **Sky Blue Theme**: Young and fun aesthetic matching the brand
- **Smooth Animations**: AOS library for scroll-triggered animations
- **6 Main Sections**:
  - Hero/Welcome
  - About Us
  - Game Library
  - Food & Beverages
  - Contact Information
  - Membership Program

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- AOS (Animate On Scroll) Library
- Font Awesome Icons
- Google Fonts (Poppins)

## Setup

1. Clone or download the project
2. Open `index.html` in a web browser
3. For GitHub Pages deployment, upload to repository and enable Pages

## Deployment

This website is ready for GitHub Pages deployment:
1. Upload all files to your GitHub repository
2. Go to Settings > Pages
3. Select source branch (main/master)
4. Your site will be available at `https://username.github.io/repository-name`
